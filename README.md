# Text font
For the project, a text font based on the *Over the Garden Wall* series was used.

![imagenfuente](https://i.imgur.com/clKRIcR.png)
```css
@font-face {
    font-family: overthegarden;
    src: url("Tomeoftheunknown-3gL3.ttf");
}

body{
    font-family: overthegarden;
}
```
